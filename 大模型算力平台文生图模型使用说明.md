# **文生图（Text\-to\-Image）使用说明**

本文档说明如何调用文生图接口。



## **1\. 接口及参数**

本项目对外暴露 OpenAI Images 兼容接口：

- `POST ``http://tops.magene.cn:11636``/api/v1/images/generations`

- 使用“用户 API Key”鉴权。

- 返回：`data[].b64_json`（Base64 图片）以及 `usage` 等字段

常用参数：

- `model` 目前支持：openai/gpt\-image\-2

- `size` 常用值：

    - `1024x1024`（square）

    - `1536x1024`（landscape）

    - `1024x1536`（portrait）

    - `2048x2048`（2K square）

    - `2048x1152`（2K landscape）

    - `3840x2160`（4K landscape）

    - `2160x3840`（4K portrait）

    - `auto`（默认）

- 自定义 `size` 限制（来自文档）：

    - 最大边长 ≤ 3840px

    - 两条边都必须是 16px 的倍数

    - 长边与短边之比 ≤ 3:1

    - 总像素数 ∈ \[655,360, 8,294,400\]

- `quality`：

    - `low` / `medium` / `high` / `auto`（默认）

- `prompt` 长度限制：约 32000 characters



## **2\. 调用方式（文生图）**

### **2\.1 cURL 示例**

```Bash
curl -X POST "http://tops.magene.cn:11636/api/v1/images/generations" \
  -H "Authorization: Bearer <你的用户APIKey>" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "openai/gpt-image-2",
    "prompt": "明媚春天，动漫",
    "quality": "low",
    "size": "1024x1024"
  }'
```



### **2\.2 响应说明**

成功响应（示例，来自文档）包含：

- `data[0].b64_json`：Base64 图片内容

- `usage`：统计信息（token、数量等）

```Bash
{
    "created": 1782203674,
    "data": [
        {
            "b64_json": "iVBORw0KGgoAAAA…………………………………………"
        }
    ],
    "background": "opaque",
    "output_format": "png",
    "size": "1402x1122",
    "quality": "low",
    "usage": {
        "total_tokens": 210,
        "input_tokens": 24,
        "output_tokens": 186,
        "input_tokens_details": {
            "text_tokens": 24,
            "image_tokens": 0
        },
        "output_tokens_details": {
            "image_tokens": 186,
            "text_tokens": 0
        }
    }
}
```



### **2\.3 Base64 解码保存（Python 示例）**

```Python
import base64
import json

resp = json.loads(open("resp.json", "r", encoding="utf-8").read())
b64 = resp["data"][0]["b64_json"]
img = base64.b64decode(b64)
with open("out.png", "wb") as f:
    f.write(img)
```
